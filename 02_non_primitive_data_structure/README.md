## 배열

같은 타입의 데이터들을 연속된 메모리 공간에 저장하는 자료구조

- 특징
  인덱스 접근
  접근 속도 빠름
  

## 연결리스트

불연속적으로 저장된 노드(Node)들이 포인터로 연결된 자료구조


- 단일 연결 리스트 (Singly Linked List)
  각 노드가 오로지 다음 노드만 가르킴

- 이중 연결 리스트 (Doubly Linked List)
  각 노드가 이전+다음 노드 가르킴

- 원형 연결 리스트 (Circular Linked List)
  마지막 노드가 다시 첫 번째 노드를 가리킴
